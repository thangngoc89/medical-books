Hướng dẫn tải sách

Hiện tại, mình đang dùng tenlua.vn để lưu trữ các file của trang web.

Bước 0. (nếu bạn không thích thì có thể bỏ qua bước này). Mở tài khoản tại tenlua.vn để thuận tiện cho việc download (không có thời gian chờ giữa các file). Để mở tài khoản bạn vào đây  . [Bấm Bắt đầu] . Chọn Facebook / Google để đăng kí cho tiện.

Bước 1. Mở link cần download, tùy thuộc vào loại tài khoản của bạn mà giao diện như hình sau


Bước 2. Bấm nút [Tải xuống chậm], chờ 1s sau đó sẽ tự động download. Sau đó, bạn chờ quá trình download kết thúc.

 

Trường hợp IDM không bắt được link, bạn mở IDM, vào Downloads => Options => chọn tab File types, kiểm tra xem phần đuôi của file có thêm vào ngay dưới dòng “Automatically start downloading the following file types” không.
